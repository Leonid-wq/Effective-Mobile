Проект запускается командой docker-compose up -d
Для проверки необходимо ввести команду curl http://localhost , в ответ на экране появится
 "Hello from Effective Mobile!"
Контейнер с самим приложением (backend) работает на порту 8080. Контейнер с nginx перенаправляет запросы с 80 порта
на хосте localhost на порт 8080, на котором они обрабатываются контейнером с приложением.
